## Fein Atelier - org
https://feinatelier.org/

Python Flaskで構築されたプログレッシブウェブアプリ（PWA）です。
基礎的なフロントエンド技術を中心に構成し、Google Cloud App Engine上で動作しています。

### サイト一覧
- [Fein Atelier - org](https://feinatelier.org/) （メインサイト）
- [fein's portal](https://portal.feinatelier.org/) （ポータルサイト）

### 概要
「Fein Atelier - org」は特定の話題に絞らない汎用的なWebサイトです。
PWAとして設計されているため、ブラウザからの利用だけでなくアプリのように動作させることも可能です。
基礎的なフロントエンド技術（HTML・CSS・JavaScript）とPythonのFlaskを組み合わせて構築しています。

#### サイト制作の背景事情
基礎的なフロントエンド技術を中心に構成しておけば、Webサイトを引っ越ししたくなっても簡単です。
特定のサイト作成サービスは利用していません。
コンテンツの所有権がサービス運営会社に帰属する場合があり、サイトが大きくなるほど移転が難しくなるためです。

#### 今後の予定
当面はHTML・CSS・JavaScriptでWebページを拡充していきます。
新しく取得した.orgドメインが定着するまでは、大規模なシステム改変は行わない予定です。
